Base de datos:

La base de datos se encuentra en el directorio bd y el fichero se llama bd_proyecto04.

Al crear la base de datos le tienes que llamar bd_proyecto04 con el cotejamiento utf8_spanish_ci

Una vez creada la base de datos copiamos el contenido del fichero bd_proyecto04 y lo pegamos en una sentencia sql dentro de la base de datos bd_proyecto04.
***************************************************************************
Acceso a la web de MyContacts.

Para acceder a la Web tienes que inciar con la pagina de login.php
